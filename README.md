## Approach

- The chatbot uses a **carefully crafted system prompt** to enforce a rude, sarcastic personality.  
- Responses are dynamically generated with **`gemini-1.5-chat`** model.  
- What makes Sarcasmobot unique is its **human-like biting humor**
